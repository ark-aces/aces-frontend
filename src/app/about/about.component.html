Ark Contract Execution Services (ACES)

ACES allows users to execute services directly from Ark transactions using the SmartData bridge field and encoded listeners services provided by ACES.

Available Services

Ethereum Contract Deployment

With this service, users are able to deploy Ethereum code contracts directly to the Ethereum network using Ark. Users supply compiled solidity compiled contract code to the service and pay for deployment using an Ark wallet.

Ethereum Transfer

With this service, users are able to send a specified amount of Eth to a target Eth account using Ark.
